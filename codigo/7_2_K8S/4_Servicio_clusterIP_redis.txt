- Práctica Servicios

- Comunicación entre un Maestro (Servidor) / Esclavo (cliente), serán dos deployments.
Para conectar los dos componentes necesitamos un servicio: de tipo ClusterIP (es el tipo
por defecto), para que el cliente pueda solicitar algo al Servidor lo tiene que hacer
a través de un servicio.
La idea es tener los datos en el maestro (Servidor) y el cliente se conectará al maestro.

Crear los dos deployments y el servicio y luego probar a conectar. El puerto que utiliza
la base de datos redis es 6379 (el puerto habra que exponerlo en el servicio)

Cuando tengamos creados los dos despliegues y el servicio, nos tenemos que conectar al POD del redis-cli
Y en la BD redis tenemos un comando llamado redis-cli con la sintaxis: redis-cli -h redis-master
-h es HOST y redis-master es el nombre del maestro (el servidor).
redis-master es como el nombre de la máquina.


## Deployment del redis-master
kubectl create deployment redis-master --image=redis

## Comprobarlo:
kubectl get deploy, pod

## El cliente Redis:
kubectl create deployment redis-cli --image=redis

## Hacer las  comprobaciones:

# El servicio: el tipo ClusterIP no sería necesario:
kubectl expose deploy redis-master --port=6379 --type=ClusterIP 

# Comprobar el servicio:
kubectl get svc

# Para conectar al redis-cli mirar el POD:
kubectl exec redis-cli-numero -it -- bash

# Dentro del contenedor redis-cli: me conecto con el maestro a través del servicio:
root@redis-cli ....# redis-cli -h redis-master 

# Al conectar, veremos el prompt: y creamos una clave-valor en la BD
redis-master:6379> set V1 10
redis-master:6379> get V1
me devuelve 10

# En una nueva ventana, entramos en el master, buscar el POD del maestro
kubectl exec -it redis-master-numero -- bash

En el prompt
root@redis-master....# redis-cli (se conecta en local)
127.0.0.1:6379> info keyspace (tengo una BD con una key)

127.0.0.1:6379> get V1 (me devuelve el valor 10)


Desde el contenedor del cliente estamos conectando con el contenedor del maestro
